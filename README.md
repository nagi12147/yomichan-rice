# Installation
1. Go to Yomichan Settings > Appearance > Configure custom CSS... > Copy and Paste my [Popup.css](https://github.com/nagi12147/yomichan-rice/blob/main/Popup.css) to Popup CSS
2. Follow my appearance settings
